How to add videos to your website

To show a video on a webpage the <video> tag is used.

The controls attribute adds play, pause and volume options.

It's always good to include width and height attributes, so that the video player loads correctly.

The <source> element allows you to specify alternative video files which the browser may choose from. The browser will use the first recognized format.

The text written between the <video> tag will only be displayed, if the browser does not support the video element.
