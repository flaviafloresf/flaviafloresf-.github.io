Table
Tag Description Usage
<table> the table data must be inside this tag used to display a table
<tr> table row used to create a new row
<td> table data used to add content to the table
<th> table header used to add a header for each column
<caption> adds caption to the table
<colgroup> Specifies a group of one or more columns in a table for formatting
<col> Specifies column properties for each column within a colgroup element
<thead> Groups the header content in a table
<tbody> Groups the body content in a table
<tfoot> Groups the footer content in a table
<colspan> used to merge columns
<rowspan> used to merge rows
