CSS Syntax

Selector Example Description
#id #firstname Selects the element with id="firstname"
.class .intro Selects the element with class="intro"
element.class p.intro Selects only <p> elements with class="intro"
* * Selects all elements
element p Selects all <p> elements
element, element, ... div, p Selects all <div> and all <p> elements

CSS Combinators

  • descendant selector (space);

    • - The following example selects all <p> elements inside <div> elements:

    div p {background-color: orange;}

    This paragraph is not descendant from the div.


  • child selector (>);

    • - The following example selects all <p> elements that are children of a <div> element:

    div > p {background-color: yellow;}

    This paragraph is not a child (but is descendant) from the div element! (inside a section element).


  • adjacent sibling selector (+);

    • - An adjacent is an element that is directly after another specific element (it means "immediately following").

    div + p {background-color: lightgreen;}

    This paragraph is directly after the div.


  • general sibling selector (~);

    • - A general sibling selects all elements that are next siblings of a specified element.

    div ~ p {background-color: lightblue;}

    This paragraph is directly after the div.
