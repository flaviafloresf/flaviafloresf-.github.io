CSS Position Property

There are five different position values:

  1. static

    2. - An element with a static position is always positioned according to the normal flow of the page:

    position: static

  2. relative

    3. - An element with a relative position is positioned relative to its normal position.

    position: relative

  3. fixed

    4. - An element with a fixed position: it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element. Ex: End of the page at the right corner.

    position: fixed
  4. absolute

    5. An element with an absolute position is positioned relative to the nearest positioned ancestor.

    If there is no positioned ancestor, it will use the document body as "ancestor".

    Note: Absolute positioned elements are removed from the normal flow, and can overlap elements.

    position: relative (ancestor)
    position: absolut;

  5. sticky

    6. An element with a sticky position is positioned based on the user's scroll position.

    I am sticky!

    In this example, element stays at the top of the page (top: 0), when you reach its scroll position.

Overflow

Overflow is used to clip the content or to add scrollbars when the content of the element is too big to fit in the specified area.

This property can have 4 values:

  1. visible: Default. The overflow is not clipped. The content renders outside the element's box.

    2. You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.




  2. hidden: The overflow is clipped, and the rest of the content will be invisible.

    3. You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.

  3. scroll: The overflow is clipped, and a scrollbar is added to see the rest of the content.

    4. You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.

  4. auto: Similar to scroll, but it adds scrollbars only when necessary.

    5. You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.

  5. overflow-x

    6. You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.
