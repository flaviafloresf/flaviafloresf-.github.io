CSS Background Properties

  • background-color: red;
  • background-image: url("paper.gif");
  • background-repeat: repeat-x; (image repeats horizontally)
  • background-repeat: repeat-y; (image repeats vertically)
  • background-attachment: scroll; (Sets whether a backgroun image is fixed or scrolls with the rest of the page)
  • background-position: left top;
    1. left
    2. left top
    3. left center
    4. right
    5. top
    6. bottom
    7. x% y%
    8. xpos ypos (in px)
    9. initial (sets this property to it's default value)
    10. inherit (inherits from the parent element)
  • (background shorthand) background: #ffffff url("img_tree.png") no-repeat right top;
    • - When using the shorthand property:
      1. background-color
      2. background-image
      3. background-repeat
      4. background-attachment
      5. background-position
