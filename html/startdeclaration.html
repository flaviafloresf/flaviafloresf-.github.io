What you need to know:

First things first:

Start declaration

To start writing a HTML code we need to write a declaration, '<!DOCTYPE html>' defines that this document is an HTML5 document.

<html> </html>

At every HTML code that we write, we should have a <html> </html> inside the <!DOCTYPE html>. Inside the <html> should also be a <head> tag.

Everything that we write at our code should be inside the <html> </html>.

How to add new pages to your website

  • First you need to create a new file in the same folder that you are currently working on. For example: "index.html"
  • Second you need to link this new file as a Hyperlink. And that's how it's made:
    • <a href="index.html" target="_self">Bring me back home</a>
    • The <a> tag (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
    • The href element is the reference specifies the URL of the page the link goes to.
    • The target element:
href element Description
_blank The link will open in a new window or tab
_self The link will open in the same frame as it was clicked (default)
_parent The link will open in the parent frame
_top The link will open in the full body of the window
framename The link will open in the named iframe
